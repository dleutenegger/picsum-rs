# picsum-rs

[![Build Status](https://github.com/dleutenegger/picsum-rs/actions/workflows/build.yml/badge.svg)](https://github.com/dleutenegger/picsum-rs/actions)
[![Crates.io](https://img.shields.io/crates/v/picsum-rs.svg)](https://crates.io/crates/picsum-rs)
[![Documentation](https://docs.rs/picsum-rs/badge.svg)](https://docs.rs/picsum-rs)

`picsum-rs` is a Rust client for the [picsum.photos API](https://picsum.photos/), allowing you to fetch random 
Placeholder images, with the possibility to customize size, blur, and color.

## Features
- Fetch random images
- Fetch specific image by its ID
- Configure image size, blur effect, black & white
- Retrieve image details

## Installation
Add `picsum-rs` to your `Cargo.toml` file:
```toml
[dependencies]
picsum-rs = "0.1"
```
